# SurvivorCore

SurvivorCore is an open-source survival game framework for OpenSimulator regions. It provides a modular system for creating immersive survival experiences, including player stats, zombie interactions, crafting, and quest systems. The framework is designed to be highly customizable and easy to use, enabling region owners to build their own survival-themed games.

## Features
- **Player Stats**: Health, Stamina, Hunger, Thirst, Coins, and more.
- **Persistent Data**: Player progress is stored on the region server.
- **Zombie AI**: Configurable non-player characters (NPCs) to challenge players.
- **Crafting and Roles**: Unlockable roles and crafting systems for progression.
- **Quest System**: Track objectives like gathering, kills, and crafting.
- **Security**: Built-in protections to prevent unauthorized modifications.

## The Last Refuge
The Last Refuge is a demo game built using SurvivorCore. Players must survive in a zombie-infested slum city and journey to a survivor island, where they can join a thriving community and take on roles to help rebuild society.
